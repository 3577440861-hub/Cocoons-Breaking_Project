## 1. 什么是 DOM？

**DOM (Document Object Model)** 是“文档对象模型”。

*   **类比**：把 HTML 网页想象成一棵“倒立的大树”。
    *   `<html>` 是树根。
    *   `<head>` 和 `<body>` 是大树枝。
    *   `<div>`、`<p>` 等是小树枝和树叶。

**为什么要知道 DOM？**
因为我们的插件就是在这棵树上“爬”，找到我们想要的“叶子”（标题）。

## 2. 变量、函数与语法速查表 (对比 Python)

虽然语言不同，但逻辑是相通的。

| 功能 | Python | JavaScript (我们用的) |
| :--- | :--- | :--- |
| **定义变量** | `x = 10` | `let x = 10;` (常用) 或 `const x = 10;` (常量) |
| **打印** | `print("Hello")` | `console.log("Hello");` |
| **定义函数** | `def foo():` | `function foo() { ... }` |
| **If 判断** | `if x > 5:` | `if (x > 5) { ... }` |
| **注释** | `# 这是注释` | `// 这是单行注释` 或 `/* 这是多行注释 */` |
| **注意** | Python 靠缩进 | JS 靠大括号 `{ }`，且句尾建议加分号 `;` |

## 3. CSS 选择器 (Selector) 怎么玩？

我们在代码里用了 `document.querySelector`，括号里的字符串就是“选择器”。

你可以把它理解为“找人的描述”。

### 常用写法：
1.  **按 Class (类名) 找**：
    *   写法：`.名字`
    *   例子：`.bili-video-card` -> 找所有 class 包含 `bili-video-card` 的元素。
2.  **按 ID 找** (唯一)：
    *   写法：`#名字`
    *   例子：`#app` -> 找 id 是 `app` 的那个元素。
3.  **组合找 (后代)**：
    *   写法：`爸爸 儿子`
    *   例子：`.bili-video-card .info-title` -> 找 `bili-video-card` 里面的 `info-title`。

### 练习：如何在浏览器里调试选择器？
1.  在 B 站页面按 **F12**。
2.  在弹出的窗口左上角，有个小鼠标图标（或者按 Ctrl+Shift+C）。
3.  用它点一下视频标题。
4.  在 Elements (元素) 面板里，你会看到一行被高亮的 HTML 代码。
5.  右键那行代码 -> **Copy** -> **Copy selector**。
6.  这就是你可以直接用到代码里的选择器！

## 4. 什么是箭头函数 (=>)？

你在代码里看到了这种写法：
```javascript
() => { ... }
```

这是 **ES6 箭头函数**，是一种简写的函数定义方式。

*   **简单理解**：`(参数) => { 做什么事 }`
*   **类比 Python**：这有点像 Python 的 `lambda` 函数，但 JS 里大量用它来做“回调”。

**为什么要用它？**
因为在 JS 里，我们经常需要“把一个函数作为参数传给另一个函数”。

**例子**：
```javascript
// setTimeout(要做的函数, 等多久)
setTimeout(
  () => { console.log("2秒到了！"); }, // 这是一个箭头函数
  2000
);
```

## 5. 异步编程：为什么代码不是一行行马上执行完？

这是 JS 最难但也最重要的概念。

**在 Python 里**：
代码是**同步**的。你一行一行写，它就一行一行执行，上一行不完，下一行绝对不动。

**在 JS (浏览器) 里**：
很多事情是**异步**的。
比如：
1.  网页加载图片需要时间。
2.  你点击按钮，系统在等你点。
3.  我们的代码不能因为等图片加载就卡死在那里。

所以 JS 发明了 **回调 (Callback)** 和 **事件循环 (Event Loop)**。

**我们代码里的体现**：
1.  `setTimeout`: “等 2 秒**之后**，再叫我。”
2.  `MutationObserver`: “我现在不干活，等网页变了**之后**，你再喊我。”

## 6. 推荐的学习路线 (作为大一学生)

如果你想系统性地学习：

1.  **HTML & CSS** (1周)：知道网页长什么样是画出来的。
2.  **JavaScript 基础** (2-3周)：变量、循环、函数、数组、对象。
3.  **DOM 操作** (1周)：也就是我们现在在玩的，用 JS 控制网页。
4.  **事件** (1周)：点击、滚动、输入等交互。
5.  **ES6+ 新特性** (1周)：箭头函数、let/const 等。

*   **MDN Web Docs** (最权威的文档，有中文版)。
